It works!

Copy/paste the following script to your website, inside the HTML's <body>, where you want comments to be shown: 

    
    <div id="comments" data-domain="{{ API_DOMAIN }}"></div> 
    <script language="javascript" type="text/javascript"> 
      var BUZZENGINE_PARAMS = { 
	  article_url:   window.location + '', 
	  article_title: document.getElementsByTagName('title').length ? 
	      document.getElementsByTagName('title')[0].innerHTML : null 
      }; 
 
      (function() { 
          var s = document.createElement('script'); 
          s.type = 'text/javascript'; 
          s.async = true; 
          s.src = 'http://{{ API_DOMAIN }}/static/js/buzzengine.min.js'; 
          var x = document.getElementsByTagName('script')[0]; 
          x.parentNode.insertBefore(s, x); 
      })();      
    </script>
And if you want a good-enough CSS stylesheet that you could further customize, copy/paste this piece to your HTML's <head>: 
    
    <link href="http://{{ API_DOMAIN }}/static/css/comments.css" 
      rel="stylesheet" type="text/css" />
