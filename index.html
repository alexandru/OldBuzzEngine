Open-source commenting system

TheBuzzEngine is open-source software that brings visitors comments to your webpage. Simple, easy to setup, runs on top of Google App Engine.

Simplicity, Control

It's an alternative to DISQUS / Facebook widgets, but you own the source-code and you can do whatever you want with the user's content.

It's also fast and scalable.

Installation

1. Get Started with Google App Engine

First download the GAE Python SDK and follow the instructions on getting started in the Google App Engine Docs.

2. Deploy TheBuzzEngine on it

Download TheBuzzEngine. You will need to modify appcfg.yml (the first line must have your App Engine application ID). You will also need to modify settings.py -- the file has comments on what to modify.

3. Copy the Javascript widget to your blog/website

Open the webpage for your App Engine instance in your browser. If you deployed correctly it will give you the Javascript widget that you need to copy/paste into your website.
